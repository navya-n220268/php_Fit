# FitCulture MongoDB Integration

## Setup Steps

1. Install MongoDB Community Server
2. Start MongoDB service
3. Install Composer
4. Run:
   composer require mongodb/mongodb
5. Create .env file:

   MONGO_URI=mongodb://127.0.0.1:27017
   DB_NAME=FitCulture

6. Run project using XAMPP
7. Access:
   http://localhost/php_fitculture/public/login.php
